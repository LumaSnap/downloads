# Support

LumaSnap is currently an early public release.

For issues, use:

```text
https://github.com/LumaSnap/downloads/issues
```

When reporting a problem, include:

- LumaSnap version.
- Windows version.
- What you clicked or pressed before the issue happened.
- Whether `LumaSnap.exe` starts and stays in the system tray.
- Any relevant error message or Windows Event Viewer entry.
